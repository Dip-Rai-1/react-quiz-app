# React Quiz App

An interactive quiz application built with React that tests your 
knowledge through timed multiple choice questions.

## Live Demo
[(https://react-quiz-app-omega-seven.vercel.app/)]

## Features
- Multiple choice questions with instant answer feedback
- Progress bar tracking quiz completion
- Points system with score tracking
- Restart quiz functionality
- Responsive design

## Tech Stack
- React
- useReducer (global state management)
- Context API (state sharing across components)
- CSS3

## What I Learned
- Managing complex state with useReducer instead of useState
- Sharing state globally using React Context API
- Component architecture and separation of concerns
- Conditional rendering based on application status

## How to Run Locally
git clone https://github.com/Dip_Rai-1/react-quiz-app
cd react-quiz-app
npm install
npm start
